本项目是基于vue2最新实战项目，vue2 +vue-resource2 + es6 +webpack 小商城
### 项目技术架构 
   * vue-cli
   * vue
   * vue-resource
   * vue-awesome-swiper
   * better-scroll
   * stylus
   * webpack
   
### 目前实现的功能
   * 广告轮播
   * 加入购物车，移除购物车
   * 显示评论 评论筛选
   * 商品详情 父子组件的通信
   
###目录结构
<pre>
├── build              // 构建服务和webpack配置
├── config             // 项目不同环境的配置
├── index.html         // 项目入口文件
├── package.json       // 项目配置文件
├── src                // 生产目录
│   ├── assets         // 图片资源
│   ├── common          // 公共的css js 资源
│   ├── components     // 各种组件
│   ├── App.vue         // 主页面 
│   └── main.js        // 预编译入口
</pre>
